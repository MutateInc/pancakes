---
title: Plan de Pruebas
---

Se definió a nivel de equipo que se van a realizar 5 tipos de pruebas. 

**Unit testing**   
**Integration testing**   
**Regression testing**   
**UAT**   
**Stress testing**  

Las pruebas que involucren el desarrollo particular de un caso de uso deberán ser diseñadas en la fase de diseño del ciclo de vida de desarrollo. En específico, esta son:   
- Unit testing
- Integration testing
- UAT
